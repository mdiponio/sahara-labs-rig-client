Installation Guide:

This document describes the steps to install and uninstall the Sahara components. This should be the first documentation to consult when setting up Sahara.

Rig Client Configuration Appendix:

This document contains a listing of the Rig Client configuration properties that are used in the common Rig Client. Properties in Rig Client extension libraries are not listed here and may be viewed in the 'Rig Client Extensions Appendix'.

Rig Client Extensions Appendix:

This document contains a list of 'actions', 'controllers' and 'runners' Rig Client extension classes contained in the Rig Client Commons library.

Development Handbook:

This document describes how to add a rig to Sahara. Installing Sahara is the first step to setting up a useful remote lab, the further steps are described here.
